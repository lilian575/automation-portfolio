**Suivi Portefeuille**

## Stack
- n8n (orchestration)
- API yahoo finance (pour les actifs)
- Gmail (notifications) 

## Fonctionnalité

Ce système envoie un mail automatique chaque jour à 00h00 pour vous envoyer le prix de chaucun de vos actif

## Résultat concret

  **Avant** : Chercher le prix sous les vastes sites de bourses trouvés les actifs qui vous corresponde
  
  **Après** : Chercher une fois les actifs puis vous avez les prix des actifs chaque jour
